# Description

Code and data for the characterize the oral bacterial microbiome in HIV-infected participants at baseline and after 24 weeks of EFV/FTC/TDF.

- Manuscript Title: Alterations in the oral microbiome in HIV-infected participants after antiretroviral therapy administration are influenced by immune status.

- Pubmed: https://www.ncbi.nlm.nih.gov/pubmed/29851662

- DOI: 10.1097/QAD.0000000000001811
